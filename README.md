## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

Frontend (port 3000):
```bash
npm run dev
```

Backend API (port 4000):
```bash
npm run server
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
